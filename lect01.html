HTML - HyperText Markup Language

The skeleton of every web page

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It uses "tags" to define elements like headings, paragraphs, images, and links.

  • HyperText: Text with links to other text (web pages).
  • Markup: Using tags to "mark up" content and give it meaning.
  • Language: A set of rules the browser understands.
الـ HTML ده زي الهيكل العظمي للصفحة. من غيره مفيش صفحة ويب. أنت بتكتب "تاجات" (علامات) والبراوزر بيفهمها ويعرضها.

Basic HTML Structure

Every HTML document follows this basic structure:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First Page</title>
  </head>
  <body>
    <!-- Your content goes here -->
    <h1>Hello World!</h1>
  </body>
</html>
<!DOCTYPE html>

Tells the browser this is HTML5.

<html>

The root element of the page.

<head>

Contains meta info (title, links, styles).

<body>

Contains the visible content.

فكر فيها كده: الـ head ده زي المعلومات اللي في بطاقتك الشخصية (اسم، معلومات)، والـ body هو أنت نفسك (اللي الناس بتشوفه).

Common HTML Tags

Tag Purpose Example
<h1> - <h6> Headings (h1 = biggest) <h1>Main Title</h1>
<p> Paragraph <p>Some text...</p>
<a> Link <a href="url">Click</a>
<img> Image <img src="photo.jpg" alt="desc">
<ul> / <ol> Lists (unordered/ordered) <ul><li>Item</li></ul>
<div> Container/Division <div>...content...</div>
<span> Inline container <span>text</span>
كل tag ليه وظيفة: h1 للعناوين الكبيرة، p للفقرات، a للروابط، img للصور. زي ما في العربية القيادة والفرامل والبنزين، كل حاجة ليها دور!

CSS - Cascading Style Sheets

The styling and beauty of web pages

What is CSS?

CSS (Cascading Style Sheets) is the language used to style and layout HTML content. It controls colors, fonts, spacing, positioning, and animations.

  • Cascading: Styles can override each other in a specific order.
  • Style: Controls appearance (colors, sizes, fonts).
  • Sheets: Can be written in separate files.
لو الـ HTML هو الهيكل العظمي، الـ CSS هو الهدوم والميكاب! بتخلي صفحتك حلوة ومرتبة ومتنسقة.

Ways to Include CSS

1. Inline CSS

Inside the element's style attribute.

<p style="color: red;">Text</p>

Not recommended for large projects

2. Internal CSS

Inside <style> in the head.

<style> p { color: red; } </style>

Good for single pages

3. External CSS

Separate .css file linked.

<link rel="stylesheet" href="style.css">

Best practice!

الأفضل دايماً تفصل الـ CSS في ملف لوحده (External)! ده بيخلي الكود منظم وسهل تعدل عليه. فكر فيها كده: كل حاجة في مكانها.

CSS Selectors

Selectors tell CSS which elements to style:

Element Selector
p { color: blue; }

Selects all <p> elements

Class Selector
.highlight { color: red; }

Selects elements with class="highlight"

ID Selector
#header { color: green; }

Selects element with id="header"

الـ Selector ده بيقول للـ CSS "روح لون ده" أو "كبر ده". ممكن تختار بالاسم (element)، أو بالكلاس (.)، أو بالـ ID (#). الفرق: الكلاس ممكن يتكرر، لكن الـ ID لازم يكون فريد!

Common CSS Properties

.card {
  /* Colors */
  color: #ffffff;              /* Text color */
  background-color: #1e293b;   /* Background */
  
  /* Typography */
  font-size: 16px;            /* Text size */
  font-family: Arial, sans-serif;
  font-weight: bold;          /* Bold text */
  
  /* Box Model */
  margin: 20px;               /* Space outside */
  padding: 15px;              /* Space inside */
  border: 1px solid #3b82f6; /* Border */
  border-radius: 10px;        /* Rounded corners */
}
The Box Model
الـ Box Model ده أهم مفهوم في CSS! كل عنصر عبارة عن صندوق: المحتوى جواه، حواليه padding (مسافة داخلية)، بعدين border (الإطار)، وبرا margin (مسافة خارجية). زي البرتقالة: اللب، القشرة البيضا، القشرة البرتقالي، والهوا حواليها!

JavaScript - The Programming Language of the Web

Adding interactivity and dynamic behavior

What is JavaScript?

JavaScript (JS) is a programming language that runs in the browser. It makes web pages interactive by responding to user actions, updating content dynamically, and communicating with servers.

  • Handle clicks and events: React when users click, type, or scroll.
  • Change content: Update the page without reloading.
  • Validate forms: Check user input before sending.
  • Animations: Create smooth visual effects.
الـ JavaScript ده اللي بيخلي الصفحة "حية"! لما تضغط على زرار ويحصل حاجة، أو لما الصفحة تتغير من غير ما ترفرش، ده كله JavaScript. بدونه الصفحة هتبقى ثابتة زي الصورة.

Variables and Data Types

Variables are containers for storing data values:

// Declaring variables
let name = "Ahmed";        // String (نص)
let age = 20;               // Number (رقم)
let isStudent = true;       // Boolean (صح/غلط)
const PI = 3.14;            // Constant (ثابت - مش بيتغير)

// Array (قائمة)
let colors = ["red", "green", "blue"];

// Object (كائن - بيانات مترابطة)
let student = {
  name: "Mohamed",
  grade: 95,
  passed: true
};
let

Variable that can change

const

Constant - cannot change

var

Old way (avoid using)

المتغير (variable) ده زي العلبة اللي بتحط فيها حاجة. let يعني العلبة دي ممكن تغير اللي جواها، const يعني مينفعش تتغير. والـ Object ده زي الملف اللي فيه بيانات مترابطة (اسم، عمر، درجات...).

Functions

Functions are reusable blocks of code that perform a specific task:

// Defining a function
function greet(name) {
  return "Hello, " + name + "!";
}

// Calling the function
let message = greet("Sara");  // "Hello, Sara!"

// Arrow function (الطريقة الحديثة)
const add = (a, b) => a + b;

console.log(add(5, 3));       // 8
الـ Function زي الماكينة: بتديها حاجة (input)، بتعمل شغلها، وبترجعلك النتيجة (output). بدل ما تكرر نفس الكود 100 مرة، اكتبه مرة واحدة في function واستخدمه!

DOM Manipulation

The DOM (Document Object Model) lets JavaScript access and change HTML elements:

// Select an element by ID
let title = document.getElementById("main-title");

// Change its content
title.textContent = "New Title!";

// Change its style
title.style.color = "red";

// Add a class
title.classList.add("highlighted");

// Select by class (returns multiple elements)
let cards = document.querySelectorAll(".card");
الـ DOM ده الطريقة اللي JavaScript بيتكلم بيها مع HTML. تخيل إن الصفحة شجرة، وكل عنصر فيها ورقة. JavaScript بيقدر يمسك أي ورقة ويغيرها!

Events

Events are actions that happen in the browser (clicks, key presses, etc.):

Common Events
  • onclick - When clicked
  • onmouseover - Mouse hover
  • onkeyup - Key released
  • onsubmit - Form submitted
  • onload - Page loaded
// Event listener
button.addEventListener("click", () => {
  alert("Button clicked!");

الـ Event ده "الحدث" اللي بيحصل. JavaScript بيسمع (listen) للأحداث دي وبينفذ كود لما تحصل. زي لما حد يدق الباب وأنت تفتحله!
