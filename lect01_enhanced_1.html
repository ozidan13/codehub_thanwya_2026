HTML Basics

The foundation of every web page. Learn how to structure content with HTML.

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using a series of elements (tags) that tell the browser how to display content.

ببساطة: الـ HTML هو لغة بنستخدمها عشان نبني صفحات الويب. بنقول للمتصفح "ده عنوان"، "ده فقرة"، "دي صورة" وهكذا.

HTML Document Structure

index.html
<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>My Page</title> </head> <body> <h1>Hello World!</h1> </body> </html>
  • <!DOCTYPE html> - Declares HTML5 document
  • <html> - Root element
  • <head> - Contains metadata (title, styles, scripts)
  • <body> - Contains visible content

CSS Basics

Make your websites beautiful with Cascading Style Sheets.

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of HTML documents. It controls layout, colors, fonts, spacing, animations, and more.

ببساطة: الـ CSS هو اللي بيخلي صفحتك جميلة. HTML بيبني الهيكل، و CSS بيلبسه وبيزوقه.

3 Ways to Add CSS

1. Inline

<p style="color:red;">Text</p>

Direct on element (not recommended)

2. Internal

<style> p { color: red; } </style>

In <head> section

3. External ✓

<link rel="stylesheet" href="style.css">

Separate file (best practice)

CSS Selectors

Selector Example Selects

Class .box { } Elements with class="box"
ID #header { } Element with id="header"
Attribute [type="text"] { } Elements with type="text"

JavaScript Basics

Add interactivity and dynamic behavior to your web pages.

What is JavaScript?

JavaScript is a programming language that makes web pages interactive. It runs in the browser and can respond to user actions, manipulate content, and communicate with servers.

ببساطة: JavaScript هو اللي بيخلي الموقع يتفاعل معاك. لما تدوس على زرار ويحصل حاجة، ده JavaScript.

Variables & Data Types

Variables
// Variables - containers for data let name = "Ahmed"; // Can be changed const PI = 3.14159; // Cannot be changed var oldWay = "avoid this"; // Old syntax // Data Types let text = "Hello"; // String let number = 42; // Number let isActive = true; // Boolean let items = [1, 2, 3]; // Array let person = { // Object name: "Ahmed", age: 25 };

Function Types
// Function Declaration function greet(name) { return "Hello, " + name; } // Function Expression const add = function(a, b) { return a + b; }; // Arrow Function (modern) const multiply = (a, b) => a * b; // Usage greet("Ahmed"); // "Hello, Ahmed" add(5, 3); // 8 multiply(4, 2); // 8

DOM Manipulation

The DOM (Document Object Model) lets JavaScript interact with HTML elements:

DOM Methods
// Selecting Elements const el = document.getElementById("myId"); const el = document.querySelector(".myClass"); const els = document.querySelectorAll("p"); // Changing Content el.textContent = "New text"; el.innerHTML = "<strong>Bold</strong>"; // Changing Styles el.style.color = "red"; el.style.backgroundColor = "#3B82F6"; // Adding/Removing Classes el.classList.add("active"); el.classList.remove("hidden"); el.classList.toggle("dark");
